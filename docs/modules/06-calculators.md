# Module: Calculators

20+ financial calculators covering EMI, SIP, tax, salary, and more. All calculators are **fully client-side** — no backend API calls, no database. Pure React with local state.

## Frontend Pages

All calculator pages live in `src/pages/` and use the `CalculatorLayout` wrapper with a `ToolsSidebar` (links to related calculators).

## Layout Pattern

```
CalculatorLayout
  └── ToolsSidebar (right — links to related calculators in same category)
  └── Calculator content (left — inputs, results, charts)
```

Calculator categories defined in `src/utils/calculators.ts`. The sidebar auto-filters to show calculators in the same category as the current page.

## No Backend

All calculators are pure client-side math:
- No API calls
- No TanStack Query
- Local React state for inputs/results
- `chart.js` used for amortization and growth charts

## SEO

- **Title:** `{CalculatorName} Calculator | Rupeepedia`
- **Description:** Generated via `src/utils/seo.ts`
- Fully indexable
- Listed in `/sitemap-calculators.xml` (generated by backend)

## Known Gaps

- No calculator-specific module doc per type — all live in a single page file per calculator
- No saving/sharing of calculator results
- No comparison between loan offers
